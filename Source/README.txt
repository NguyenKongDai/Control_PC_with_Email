Các lệnh chương trình nhận sẽ thực hiện, lệnh ở phần nội dung
screenshot : chụp màn hình
shutdown : tắt máy trong 10s
reset : khởi động lại máy trong 10s
list apps : gửi về danh sách các ứng dụng đang chạy
list processes : gửi về danh sách các chương trình đang chạy
kill <pid> hoặc top <pid> : stop/kill app/process bằng id, id có thể xem thông qua lệnh list apps/processes
keylogger hook : bắt đầu nhận phím nhấn
keylogger unhook : dừng việc nhận phím nhấn, gửi các phím nhấn trở lại
capturevideo <time> : quay video cam, mặc định 10s nếu không có trường <time?
copyfile <addr1> <addr2> : copy file từ addr1 đến addr2 
quit : thoát quá trình kết nối
